# Section Library

## Families

| Family | Role |
|--------|------|
| Banners | Hero / top-of-page blocks |
| Text blocks | Editorial / long-form text sections |
| Cards | Grid-based content (image, icon, pricing, steps, split) |

## Card section keys

- `cards_image_grid`
- `cards_icon_grid`
- `cards_icon_split_grid`
- `cards_pricing_grid`
- `cards_steps_grid`

## Pattern (all sections)

- `settings` — layout, spacing, theme, width, columns, align, etc.
- `fields` — copy, media, icons, card repeaters.
- `template` — PHP partial slug.
- `css` — section block in bundle.
