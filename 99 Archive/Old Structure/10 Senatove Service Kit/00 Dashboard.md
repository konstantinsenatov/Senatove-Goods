# Senatove Service Kit Dashboard

## Project Overview
Senatove Service Kit is a dedicated WordPress-first product subproject inside Senatove Goods.
The first release is a premium service-focused WordPress stack:
- Theme: **Senatove Service Theme** (`senatove-service-theme`)
- Plugin: **Senatove Sections Builder** (`senatove-sections-builder`)

The delivery model is **Local + WordPress + Cursor** with a controlled section-building workflow in admin and premium frontend rendering in theme.

## Current Status
- Subproject workspace created and ready for structured execution.
- Core product direction fixed: premium service WordPress theme + companion curated builder plugin.
- Documentation baseline initialized for implementation kickoff.
