# Theme Spec

## Public Name
**Senatove Service Theme**

## Slug
`senatove-service-theme`

## Role
Frontend presentation engine for the product: transforms structured section data into a premium service-site experience.

## Visual Direction
- Premium, minimal, calm, warm.
- Editorial-influenced typography and spacing rhythm.
- Intentional whitespace and restrained accent usage.
- Strong readability and conversion-focused hierarchy.

## Core Page Templates (v1)
- Home / Landing template.
- Standard Service Page template.
- About / Studio narrative template.
- Contact-focused template.
- Default page fallback template with section rendering support.

## What Must Be Editable
- Section content fields (headlines, body copy, media, CTA labels/links, FAQ items, testimonial entries).
- Controlled style variants exposed by plugin-defined options.
- Section ordering via builder stack.

## What Should Stay Fixed
- Core design tokens (type scale, spacing scale, color system).
- Baseline layout grid and container behavior.
- Component-level visual integrity rules.
- Animation/motion discipline and interaction quality.

## Product Positioning
A premium service-theme system with curated composition: visually refined enough for high-end client-facing sites, operationally simple enough for non-technical teams.
